# 类加载机制
## 类加载运行全过程
当我们用java命令运行某个类的main函数启动程序时，首先需要通过类加载器把主类加载到 JVM。
```java
public class HelloWorld {
    public static final int initValue = 88;

    public int cal() {
        int m =5;
        int n = 4;
        int p = (m+n)* 100;

        return p;
    }

    public static void main(String[] args) {
        
        new HelloWorld().cal();
    }
}
