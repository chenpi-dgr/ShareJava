## 叨叨
你是否看见框架原理就兴奋？你是否看见源码原理的文章就想点进去看看？但是最终的效果都不尽人意。博主以前对这种情况也深有体会。为什么会这样？
- 流量为先，短短的5-10分钟很难有一些深层次的体会。
- 输入的东西不系统，东一榔头西一棒槌，学习不成体系。

那么今天janker就为你带来较为系统的 MyBatis 3.x 的源码原理之旅。
